# 构建

## 前端

*从Vue.js构建* <https://github.com/little1d/Spark_fe>

1. 准备依赖库
   `npm install`
2. 检查开发模式
   `npm run-script dev`
3. 构建
   `run run-script build`

## 后端

*SpringBoot项目* <https://github.com/kites262/SparkSports>

1. 打包
   `mvn package`
